Beauty & Wellness Appointment Booking Website

A simple and elegant salon appointment booking system built using HTML, CSS, JavaScript, and Firebase Firestore.
Users can book services like Facial, Spa, Waxing, Hair Massage, and Hydrafacial, and receive instant confirmation after booking.

 Features

Real-time booking with Firebase Firestore
Instant success message after appointment booking
Admin can view all bookings in Firebase console
Responsive and easy-to-use design

 Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Firebase Firestore
Hosting: Firebase Hosting

 How It Works

Click “Book Appointment”
Choose a service
Confirm booking → Data stored in Firestore
Message shown: “Your appointment is booked! Contact us for any query.”
